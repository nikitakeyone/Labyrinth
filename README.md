# Labyrinth - Pathfinding Algorithms Visualization

![A* Solver Example](https://github.com/k1noX/Labyrinth/blob/master/media/image.gif)

Visualization of some algorithms that could help you find the way out of the maze by searching the shortest path from
source to destination!

## Getting Started

### Dependencies

The project uses **PyQt5** to build the grid and animate the solving process.

### Installation and Running

Install **requirements.txt**:
```
pip install -r requirements.txt
```

Then simply run **main.py**:
```
cd ./src
python3 ./main.py
```
